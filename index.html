Desafios propostos

Desafio dos Emojis
O desafio do emoji consistia em fazer alguns emojis com base em códigos hexadecimais.
Desafio das Imagens
O Desafio das imagens consistia em fazer a anexagem de várias imagens de forma diferente na página.
Desafio do Mapa Mundi
O Desafio do Mapa mundi consistiu na remontagem de nove imagens que estavam dispostas aleatoriamente.
Desafio das Mensagens
O Desafio das Mensagens consistia na cofecção de uma estrutura para mostrar a função de cada tag de título.
Desafio das mídias sociais
ODesafio das mídias sociais consiste em uma uma pasta ondee estão dispostas as formas de contato para o professor do curso.
Desafio das Tags
O Desafio das Tags consiste em uma lista com uma série de estruturas abreviadas.
